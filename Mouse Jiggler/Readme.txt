MouseJiggler Payload 

_______________________________________________________________________________

Description & Notes:


- a makeshift mouse-jiggler that enters the capslock key every 10 seconds to make sure the screen doesnt freeze and it looks as though you are active.

- you can change the time between clicks by editing the DELAY in the payload.

- for windows only.
